## **Project Setup**  

### **Local Setup**  

1. **Clone the repository**  
   ```sh
   git clone https://github.com/your-username/brightcity-events.git
   cd brightcity-events
   ```

2. **Start the backend**  
   ```sh
   cd server  
   npm install  
   npm run start:dev  
   ```

3. **Start the frontend**  
   ```sh
   cd client  
   npm install  
   npm start  
   ```

4. **Check the API Documentation**  
   Open in your browser [http://localhost:5000/api/docs](http://localhost:5000/api/docs) (Swagger)
